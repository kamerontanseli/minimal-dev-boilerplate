# Minimal Dev Boilerplate

Inspired by [A better front end experience](https://medium.com/choojs/a-better-frontend-experience-7b0498c85658) and [Go Buildless](https://css-tricks.com/going-buildless/).

### Tools Provided

* [Choo](https://choo.io)
* [Import Maps](https://github.com/WICG/import-maps)
* [PWA/Service Works](https://developers.google.com/web/progressive-web-apps/)
* [Pika](https://pika.dev)
* [ESLint](https://eslint.org)
* [Prettier](https://prettier.io)
